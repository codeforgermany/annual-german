Code for Germany möchte drei Fragen beantworten: Wie können Code und Software dabei helfen, Stadt und Nachbarschaft zu verbessern? Wie können kollaborative Ansätze mit Hilfe von neuen Technologien zu nachhaltigen Veränderungen führen? Wie können städtische Regierungen externes Know-How einsetzen um Bürgerzentrierter, transparenter, und effizienter zu werden?

Durch die Beantwortung dieser Fragen möchte Code for Germany zu der Entwicklung einer IT-basierten Regierungs- und Verwaltungskultur beitragen, die ihren Bürgern digitale Dienste bieten kann. Durch die Vertiefung der Zusammenarbeit zwischen Stadtregierungen und technikaffinen lokalen Gemeinschaften erhofft sich Code for Germany die präservierung nachhaltiger Beziehungen zwischen möglichst vielen ansäßigen Interessengruppen. Um dies zu erreichen hat Code for Germany vier verschiedene Module entwickelt, die strategisch aufeinander aufbauen. Zusammen resultieren diese in einem größeren, zivilgesellschaftlichen Innovationsprozess.

Zum Schluss möchten wir uns noch ausführlich bei all jenen bedanken, die dabei geholfen haben Code for Germany zu initiieren und das Prohekt über das letzte Jahr hinweg unterstützt haben. Wir freuen uns sehr auf weitere erfolgreiche Zusammenarbeit um Jahr 2015.

Das Code for Germany Team
